#xrf-gui
This is a GUI based program that allows for the qualitative and quantitative analysis of x-ray fluorescense data.
In order to use this program, follow below.

1. Refer to the README file for the PyXRF program at https://github.com/NSLS-II/PyXRF. This sets up an environment and downloads the required prequisite code that this program uses.

2. Obtain all the code from the xrfGui folder and run the 'implementation.ipynb' file and run each cell to use the program.
